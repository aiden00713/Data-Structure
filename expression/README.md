## 中序式計算

A. 能輸入中序式轉成後序式。<br>
B. 能將後序式加以運算求值。<br>
C. 運算子可以有 +, - , *, /, ^ (次方)五種。<br>
D. 運算元由A~Z來表示，其對應之值如下陣列之宣告順序:
int op_value[26]={10,30,6,9,8,11,12,13,7,8,22,56,77,76,55,56,43,40,13,19,18,16,46,52,61,63};

----

第一列輸入測試案例的個數T。緊接著是每個測試案例的資料。<br>
每個測試案例的輸入為一個字串，字串的長度小於30。

----

|Sample Input   |Sample Output |
| --- | --- |
|2| |
|A+B/C|ABC/+<br>15|
|X*Y+Z|XY*Z+<br>3235|
